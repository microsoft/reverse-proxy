Class DelayingQueueBase<TItem>

Class DelayingQueueBase is a delegating base class for IDelayingQueue<TItem> interface. These classes are ported from go, which favors composition over inheritance, so the pattern is followed. Implements the WorkQueueBase<TItem>. Implements the IDelayingQueue<TItem>.
