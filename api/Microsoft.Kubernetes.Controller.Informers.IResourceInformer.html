ReadyAsync(CancellationToken)

Returns a task that can be awaited to know when the initial listing of resources is complete. Once an await on this method it is safe to assume that all of the knowledge of this resource type has been made available, and everything going forward will be updatres.

Register(ResourceInformerCallback<IKubernetesObject<V1ObjectMeta>>)

Registers a callback for change notification. To ensure no events are missed the registration may be created in the constructor of a dependant . The returned registration should be disposed when the receiver is ending its work.
